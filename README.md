# Projeto_Basquete
Projeto de criação de uma tabela de acompanhamento de jogos de basquete para Maria.

Primeiro passo é instalar o Python no seu computador, fazendo o download no link https://www.python.org/downloads/release/python-386/ e instalando a versão compatível com o seu sistema operacional. Caso for o Windows, é possível ver um tutorial neste site https://python.org.br/instalacao-windows/

Importante, é necessário marcar a opção "Add Python"

Segundo passo é instalar a biblioteca PyQt5 no computador inserindo no cmd o seguinte comando: pip install PyQt5

Terceiro passo é executar o arquivo main.py e utilizar o aplicativo.
